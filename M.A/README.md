# Proyecto Web Estática - Plantilla mínima

Plantilla básica de sitio web estático con HTML, CSS y JavaScript mínimos.

Pasos para usar:

1. Abrir `index.html` en el navegador.
2. Editar `css/styles.css` y `js/script.js` según necesidades.

Opcional: servir localmente con un servidor estático (ej. `npx http-server` o abrir con Live Server en VS Code).
